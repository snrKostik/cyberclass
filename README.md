# Minecraft Cyber Tournament System

Локальная веб-система для проведения Minecraft-турниров с автоматической турнирной сеткой, админ-панелью и режимом отображения результатов для проектора.

---

## О проекте

Minecraft Cyber Tournament System — это MVP-система для организации локальных и школьных Minecraft-турниров без необходимости разворачивать сложную инфраструктуру.

Система автоматизирует:

* создание матчей;
* отображение турнирной сетки;
* ведение лидерборда;
* управление турниром;
* отображение результатов на проекторе.

Проект ориентирован на простоту запуска и минимальные требования к окружению.

---

## Возможности

### Турнирная система

* Single Elimination (олимпийская система)
* Автоматическая генерация сетки
* Автоматическое продвижение победителя
* Финальный лидерборд

### Админ-панель

* Управление матчами
* Управление командами
* Редактирование результатов
* Контроль состояния турнира

### Display / Projector Mode

* Отображение турнирной сетки
* Выделение победителей
* Отображение Top-3 команд
* Режим для проекторов и больших экранов

### Таймер между матчами

* Автоматический таймер после завершения матча
* Подготовка следующей карты
* Пауза между дисциплинами
